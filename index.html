1. Create a Document in GitHub Go to githubdesktop Click New Repository Give it a name → e.g., my-project. Click Create repository. To add another file: Click Add file → Create new file Name it (e.g., notes.md) Add your content Click Commit changes 2. Add a Collaborator to a GitHub Repo Go to your repository on GitHub. Click Settings (top menu). In the left sidebar, select Collaborators & teams (or Manage access). Click Add people. Enter their GitHub username or email. Click Add collaborator. 3. Commit, Push, Pull Setup Clone repository Commit changes (save them locally) Push changes (upload to GitHub) Pull changes (download latest from GitHub) 4. Make GitHub Pages Go Live Step 1 — Prepare your repo You can use: A simple index.html file Or Markdown (GitHub auto-renders) Step 2 — Enable GitHub Pages Go to your repo on GitHub. Click Settings. Scroll to Pages in the left menu. Under Source, select: Deploy from branch Branch: main Folder: /root (or /docs) Click Save.
